Fuentes de datos compatibles

Archivos CSV

Nuestro plugin de comparación de productos te permite importar datos desde múltiples fuentes y crear comparativas detalladas para tus clientes. Compatible con CSV, bases de datos, Apios y web scraping.

  • Importación automàtica
  • Detección de columnas
  • Validación de dades

Bases de datos

Conecta directamente con tu base de datos y ejecuta consultas SQL personalizadas para obtener exactamente los datos que necesitas.

  • Consultas SQL personalizadas
  • Conexión segura
  • Actualización automática

API Endpoints

Integra datos de cualquier API REST. Configura tus peticiones HTTP y mapea las respuestas a los campos de comparación.

  • Soporte REST y GraphQL
  • Autenticación flexible
  • Transformación de datos

Web Scraping

Extrae datos directamente de tu tienda en línea u otras páginas web. Nuestro scraper inteligente se adapta a la estructura de tu sitio Web.

  • Scraping inteligente
  • Selectores personalizables
  • Programación automática

Como funciona el proceso

1

Seleciona la fuente de datos

Elige entre CSV, base de datos, API o web scraping según tus necesidades.

2

Configura la conexión

Proporciona las credenciales necesarias o carga el fichero. El sistema validará la conexión automáticamente.

3

Mapea los campos

Asigna los campos de tus datos en los campos de comparación: nombre, precio, características, etc.
